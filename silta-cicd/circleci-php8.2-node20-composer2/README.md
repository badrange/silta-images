# silta-circleci
A docker image used circleCI, based on `cimg/php:8.2.13-node` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm

## Versions
- PHP: 8.2.13
- Composer: 2.5.1
- Node: 20.9.0
- Helm: v3.6.3
