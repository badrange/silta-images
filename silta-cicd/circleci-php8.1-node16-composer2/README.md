# silta-circleci
A docker image used circleCI, based on `cimg/php:8.1.7-node` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm

## Versions
- PHP: 8.1.7
- Composer: 2.1.12
- Node: 16.15.1
- Helm: v3.13.3
