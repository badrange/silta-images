# silta-circleci
A docker image used circleCI, based on `cimg/php:8.1.23-node` with the following additions:

- Composer configured correctly
- Drush-launcher and coder pre-installed
- Vim, useful for debugging
- kubernetes and helm

## Versions
- PHP: 8.1.23
- Composer: 2.5.1
- Node: 18.17.1
- Helm: v3.14.0
