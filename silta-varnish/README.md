# Silta Varnish docker image for Drupal projects

Docker image based on official varnish image.

Varnish 6.x image specifics:

- based on Debian
- runs on port 80
- runs as root

Varnish 7.x image specifics:

- based on Alpine
- runs on port 8080
- runs as varnish

## Usage

docker pull wunderio/silta-varnish.
